## Version 1.19.9 Release Notes

### New Features

- [ NEW ] Added support for passing parameters through URLs for all tool types in Tool Plaza.
- [ NEW ] Added a parameter visibility setting for Data Query tools. Hidden parameters are passed to queries in the background.

### Optimizations

- [ OPTIMIZATION ] Improved column widths and information display in the Scene Management list under Platform Management.

### Fixed

- [ FIXED ] Fixed an issue where invalid sorting parameters prevented the Scene Management page under Platform Management from opening.
- [ FIXED ] Fixed Windows compatibility issues with Excel exports and analysis report exports.
