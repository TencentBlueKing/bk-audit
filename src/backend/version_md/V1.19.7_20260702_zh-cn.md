## V1.19.7 版本更新日志

### 新增

- [ 新增 ] 策略运行状态可观测接口，支持查询离线和实时策略任务运行状态
- [ 新增 ] 场景隔离支持计算平台个人数据表授权管理
- [ 新增 ] 风险智能搜索与智能分析新增 AI 可观测能力，提升分析、导出等任务的运行状态追踪能力
- [ 新增 ] 历史分析报告及导出任务支持失败重试

### 优化

- [ 优化 ] 智能搜索体验优化，支持全选后清除勾选，优化表格筛选与搜索条件联动，分析报告列表支持字段筛选
- [ 优化 ] 用户画像游戏关联列表样式、搜索组件和操作列展示，导出结果补充账号列
- [ 优化 ] 工具管理操作列展示，个人数据表不展示无权限系统

### 修复

- [ 修复 ] 风险列表初始化缺少时间范围、风险命中策略展示多余 ID 的问题
- [ 修复 ] 新建策略步骤条样式重复、编辑策略名称触发策略重启的问题
- [ 修复 ] 智能分析正式响应中残留思考占位的问题
- [ 修复 ] 资产同步状态异常问题
