## V1.17.0 版本更新日志

### 新增

- [ 新增 ] 审计日志与资产添加 Doris 存储支持，支持高效灵活的下钻分析能力。
- [ 新增 ] 审计日志检索支持通过 Doris 查询。
- [ 新增 ] 审计快照数据链路代码梳理优化。
- [ 新增 ] 策略审计: 开放其他/资产数据的实时计算支持
- [ 新增 ] 默认启用 APM（应用性能管理），更完善地监控系统运行状态。

### 优化

- [ 优化 ] Python 版本升级至 3.11，Django 版本升级至 4.2。
- [ 优化 ] 手动执行处理套餐兼容无参数情况。
- [ 优化 ] 增加风险处理任务调度频次，减少处理延时。
