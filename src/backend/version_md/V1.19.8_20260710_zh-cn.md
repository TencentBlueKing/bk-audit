## V1.19.8 版本更新日志

### 新增

- [ 新增 ] 平台级工具与资源统一管理，支持工具上下架、工具广场展示控制，并支持按场景、系统配置资源可见范围与授权共享
- [ 新增 ] 平台级工具支持按场景、系统设置可见范围，并支持按可见范围覆盖参数默认值
- [ 新增 ] 资产同步新增可观测性指标上报，支持同步健康度、数据量差异和异常事件监控

### 优化

- [ 优化 ] 代币元字段为空时，工具广场搜索关键词文案由“未设汇率”调整为“未设代币兑换比例”
- [ 优化 ] 策略列表加载逻辑，降低标签接口异常或组件未就绪导致列表不展示的概率
- [ 优化 ] 场景信息关联系统和关联数据表分批加载，提升大数据量场景下的页面打开体验
- [ 优化 ] 离职人员搜索和标签配色，区分离职人员与正常搜索命中人员
- [ 优化 ] 平台管理工具和报表列表筛选、展示与操作体验

### 修复

- [ 修复 ] 处理规则字段映射不显示的问题
- [ 修复 ] BKVision 工具编辑变量时“使用默认值”复选框无法取消勾选的问题
- [ 修复 ] BKVision 图表重复渲染的问题
- [ 修复 ] IAMV4 策略字段值关联孤儿系统时的异常
